# iphone-forensics

## Install

Requires [Python 3.4](https://www.python.org/) or higher.

# Run
Ideal dependencies:
```bash
pip install plistlib
```
Or
```bash
pip install - r requirements.txt
```

Run
```py
python run.py
```
